# Operator Guide

## First Deployment
1. Register operator, production lines, design-authority/type-
   approval scope, staff and robots.
2. Import existing production and quality-record history.
3. Run read-only design-authority-scope and fabrication/welding/
   inspection robot mission dry-runs.
4. Configure safety-class allowed sets and human sign-off paths.
5. Publish a dry-run release record and audit export.

## Minimum Production Controls
- design-authority/type-approval-scope validation before any
  production run
- governor gate on every robot action before dispatch
- human sign-off for :high/:safety-critical actions (a delivery
  release without a completed type-approval inspection, a
  safety-critical design-parameter change)
- evidence-backed release records
- audit export for every dispatch, sign-off and release
- backup manual design-authority review

## Certification
Certified operators must prove robot-safety integrity, design-
authority discipline, evidence-backed release records and human
review for release-affecting actions.
